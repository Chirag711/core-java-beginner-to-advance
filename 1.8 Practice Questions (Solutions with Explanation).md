## 1. Print your name, age, and city using variables

### Code

```java
public class PersonalInfo {
    public static void main(String[] args) {
        String name = "Chirag";
        int age = 28;
        String city = "Pune";

        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("City: " + city);
    }
}
```

### Explanation

1. `String name` → stores text (non-primitive data type)
2. `int age` → stores integer value
3. `System.out.println()` → prints output to console
4. `+` operator → used for string concatenation

## 2. Perform all arithmetic operations on two numbers

### Code

```java
public class ArithmeticOperations {
    public static void main(String[] args) {
        int a = 20;
        int b = 10;

        System.out.println("Addition: " + (a + b));
        System.out.println("Subtraction: " + (a - b));
        System.out.println("Multiplication: " + (a * b));
        System.out.println("Division: " + (a / b));
        System.out.println("Modulus (Remainder): " + (a % b));
    }
}
```

### Explanation

1. `+` → Addition
2. `-` → Subtraction
3. `*` → Multiplication
4. `/` → Division (integer division)
5. `%` → Remainder

Parentheses `( )` are used to ensure correct calculation before concatenation.

## 3. Convert double to int using type casting

### Code

```java
public class TypeCastingExample {
    public static void main(String[] args) {
        double number = 10.75;

        int result = (int) number;

        System.out.println("Original double value: " + number);
        System.out.println("Converted int value: " + result);
    }
}
```

### Explanation

1. `double number = 10.75` → decimal value
2. `(int) number` → explicit casting (narrowing)
3. Decimal part is removed → result becomes `10`

## 4. Check whether a number is positive or negative

### Code

```java
public class PositiveNegative {
    public static void main(String[] args) {
        int num = -5;

        if (num > 0) {
            System.out.println("Positive number");
        } else if (num < 0) {
            System.out.println("Negative number");
        } else {
            System.out.println("Zero");
        }
    }
}
```

### Explanation

1. `if (num > 0)` → checks positive
2. `else if (num < 0)` → checks negative
3. `else` → handles zero
4. Conditional statements control program flow

## 5. Find remainder of two numbers

### Code

```java
public class RemainderExample {
    public static void main(String[] args) {
        int a = 17;
        int b = 5;

        int remainder = a % b;

        System.out.println("Remainder: " + remainder);
    }
}
```

### Explanation

1. `%` operator gives remainder
2. `17 % 5 = 2`
3. Useful in problems like even/odd, cyclic operations
