
# 1. Java Basics (Variables, Data Types, Operators)

## 1.1 Introduction to Java Program

Java is an object-oriented programming language used to build platform-independent applications.

### Basic Structure of Java Program

```java
class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

### Explanation

1. `class Main` → Class definition
2. `main()` → Entry point of program
3. `System.out.println()` → Prints output

---

## 1.2 Variables in Java

A variable is used to store data.

### Syntax

```java
dataType variableName = value;
```

### Example

```java
public class VariablesDemo {
    public static void main(String[] args) {
        int age = 25;
        String name = "Chirag";
        
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
    }
}
```

### Practice

```java
// Try changing values
int salary = 50000;
String company = "TCS";
System.out.println(company + " " + salary);
```

## 1.3 Types of Variables

### 1. Local Variable

```java
public class LocalVar {
    public static void main(String[] args) {
        int x = 10; // local variable
        System.out.println(x);
    }
}
```

### 2. Instance Variable

```java
class Student {
    int marks = 90; // instance variable
}
```

### 3. Static Variable

```java
class Demo {
    static int count = 0;
}
```

## 1.4 Data Types in Java

### 1. Primitive Data Types

| Type    | Example |
| ------- | ------- |
| int     | 10      |
| float   | 10.5f   |
| double  | 20.99   |
| char    | 'A'     |
| boolean | true    |

### Example

```java
public class DataTypesDemo {
    public static void main(String[] args) {
        int a = 10;
        float b = 20.5f;
        char c = 'A';
        boolean isJavaFun = true;

        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
        System.out.println(isJavaFun);
    }
}
```

## 1.5 Type Casting

### 1. Implicit Casting (Widening)

```java
public class CastingDemo {
    public static void main(String[] args) {
        int a = 10;
        double b = a; // automatic

        System.out.println(b);
    }
}
```

### 2. Explicit Casting (Narrowing)

```java
public class CastingDemo {
    public static void main(String[] args) {
        double a = 10.5;
        int b = (int) a;

        System.out.println(b);
    }
}
```

## 1.6 Operators in Java

### 1. Arithmetic Operators

```java
public class ArithmeticDemo {
    public static void main(String[] args) {
        int a = 10, b = 5;

        System.out.println(a + b); // 15
        System.out.println(a - b); // 5
        System.out.println(a * b); // 50
        System.out.println(a / b); // 2
        System.out.println(a % b); // 0
    }
}
```

---

### 2. Relational Operators

```java
public class RelationalDemo {
    public static void main(String[] args) {
        int a = 10, b = 20;

        System.out.println(a > b);
        System.out.println(a < b);
        System.out.println(a == b);
        System.out.println(a != b);
    }
}
```

---

### 3. Logical Operators

```java
public class LogicalDemo {
    public static void main(String[] args) {
        boolean x = true, y = false;

        System.out.println(x && y);
        System.out.println(x || y);
        System.out.println(!x);
    }
}
```

---

### 4. Assignment Operators

```java
public class AssignmentDemo {
    public static void main(String[] args) {
        int a = 10;

        a += 5; // a = 15
        a -= 3; // a = 12

        System.out.println(a);
    }
}
```

---

### 5. Unary Operators

```java
public class UnaryDemo {
    public static void main(String[] args) {
        int a = 10;

        System.out.println(++a); // 11
        System.out.println(a--); // 11
        System.out.println(a);   // 10
    }
}
```

## 1.7 Hands-on Mini Practice

### Program 1: Simple Calculator

```java
public class Calculator {
    public static void main(String[] args) {
        int a = 20, b = 10;

        System.out.println("Addition: " + (a + b));
        System.out.println("Subtraction: " + (a - b));
        System.out.println("Multiplication: " + (a * b));
        System.out.println("Division: " + (a / b));
    }
}
```

---

### Program 2: Swap Two Numbers

```java
public class Swap {
    public static void main(String[] args) {
        int a = 5, b = 10, temp;

        temp = a;
        a = b;
        b = temp;

        System.out.println("a: " + a + ", b: " + b);
    }
}
```

---

### Program 3: Check Even or Odd

```java
public class EvenOdd {
    public static void main(String[] args) {
        int num = 7;

        if (num % 2 == 0)
            System.out.println("Even");
        else
            System.out.println("Odd");
    }
}
```
