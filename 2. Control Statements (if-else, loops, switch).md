# 2. Control Statements (if-else, loops, switch)

## 2.1 Introduction

Control statements are used to control the flow of execution in a program. They help in decision-making and repeating tasks.

There are three main types:

1. Decision Making (if, if-else, switch)
2. Looping (for, while, do-while)
3. Jump statements (break, continue)

---

## 2.2 if Statement

Used to execute a block of code only if a condition is true.

### Syntax

```java
if (condition) {
    // code
}
```

### Example

```java
public class IfExample {
    public static void main(String[] args) {
        int age = 20;

        if (age >= 18) {
            System.out.println("Eligible to vote");
        }
    }
}
```

### Explanation

* Condition is checked (`age >= 18`)
* If true → block executes
* If false → skipped

---

## 2.3 if-else Statement

Used when you have two conditions.

### Syntax

```java
if (condition) {
    // true block
} else {
    // false block
}
```

### Example

```java
public class IfElseExample {
    public static void main(String[] args) {
        int num = 5;

        if (num % 2 == 0) {
            System.out.println("Even");
        } else {
            System.out.println("Odd");
        }
    }
}
```

---

## 2.4 else-if Ladder

Used when multiple conditions are present.

### Example

```java
public class GradeExample {
    public static void main(String[] args) {
        int marks = 75;

        if (marks >= 90) {
            System.out.println("Grade A");
        } else if (marks >= 75) {
            System.out.println("Grade B");
        } else if (marks >= 50) {
            System.out.println("Grade C");
        } else {
            System.out.println("Fail");
        }
    }
}
```

---

## 2.5 Nested if

if inside another if.

```java
public class NestedIf {
    public static void main(String[] args) {
        int age = 20;
        boolean hasLicense = true;

        if (age >= 18) {
            if (hasLicense) {
                System.out.println("Can drive");
            }
        }
    }
}
```

## 2.6 switch Statement

Used for multiple conditions based on a single variable.

### Syntax

```java
switch (variable) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // code
}
```

### Example

```java
public class SwitchExample {
    public static void main(String[] args) {
        int day = 3;

        switch (day) {
            case 1:
                System.out.println("Monday");
                break;
            case 2:
                System.out.println("Tuesday");
                break;
            case 3:
                System.out.println("Wednesday");
                break;
            default:
                System.out.println("Invalid day");
        }
    }
}
```

### Explanation

* `switch` checks value
* `case` matches value
* `break` stops execution
* `default` runs if no match

# 2.7 Loops in Java

Loops are used to repeat a block of code.

---

## 2.7.1 for Loop

### Syntax

```java
for (initialization; condition; update) {
    // code
}
```

### Example

```java
public class ForLoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }
    }
}
```

### Explanation

1. `int i = 1` → start
2. `i <= 5` → condition
3. `i++` → increment

---

## 2.7.2 while Loop

### Syntax

```java
while (condition) {
    // code
}
```

### Example

```java
public class WhileExample {
    public static void main(String[] args) {
        int i = 1;

        while (i <= 5) {
            System.out.println(i);
            i++;
        }
    }
}
```

---

## 2.7.3 do-while Loop

Executes at least once.

### Syntax

```java
do {
    // code
} while (condition);
```

### Example

```java
public class DoWhileExample {
    public static void main(String[] args) {
        int i = 1;

        do {
            System.out.println(i);
            i++;
        } while (i <= 5);
    }
}
```

# 2.8 break and continue

### break Example

```java
public class BreakExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            if (i == 3) {
                break;
            }
            System.out.println(i);
        }
    }
}
```

### continue Example

```java
public class ContinueExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            if (i == 3) {
                continue;
            }
            System.out.println(i);
        }
    }
}
```

# 2.9 Hands-on Programs

## Program 1: Check Largest Number

```java
public class LargestNumber {
    public static void main(String[] args) {
        int a = 10, b = 20, c = 15;

        if (a >= b && a >= c) {
            System.out.println("A is largest");
        } else if (b >= a && b >= c) {
            System.out.println("B is largest");
        } else {
            System.out.println("C is largest");
        }
    }
}
```

## Program 2: Print Table of Number

```java
public class Table {
    public static void main(String[] args) {
        int num = 5;

        for (int i = 1; i <= 10; i++) {
            System.out.println(num + " x " + i + " = " + (num * i));
        }
    }
}
```

## Program 3: Menu Driven Program using switch

```java
public class MenuProgram {
    public static void main(String[] args) {
        int choice = 2;
```
